# Sistema de Gestión de Citas - Centro de Psicología

Este proyecto es una aplicación web desarrollada con Django para la **gestión de citas en un centro de psicología**. Está diseñado para permitir que recepcionistas agenden, visualicen y gestionen citas entre pacientes y psicólogos.

Funcionalidades principales

- Registro y autenticación de usuarios personalizados (recepcionistas y psicólogos).
- Registro de pacientes con sus datos básicos.
- Agendado de citas asociadas a un paciente y un psicólogo.
- Visualización de todas las citas registradas.
- Estados de citas: `Agendada`, `Asistida`, `Cancelada`.
- Panel de administración con control completo de usuarios y registros.
